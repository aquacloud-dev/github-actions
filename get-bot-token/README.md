
# Get bot token
> Authenticate Github App

## Inputs 

| Name | Description | Default | Required | 
| ---- | ----------- | ------- | -------- |
| `BOT_ID` | The ID of the github-app | `nd` | `true` |
| `BOT_PEM` | The bot PEM in base64 | `nd` | `true` |


## Outputs 

| Name | Description |
| ---- | ----------- |
| `token` | Github Token |
